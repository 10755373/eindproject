Links to

The data is retrieved from www.kaggle.com as a csv-file (https://www.kaggle.com/russellyates88/suicide-rates-overview-1985-to-2016)

Regarding the worldmap: for drawing it, retrieving data from the json file and appending colors to countries, the example from Mark Markoh is used (see: https://github.com/markmarkoh/datamaps/blob/master/src/examples/highmaps_world.html).

Regarding the linegraph: for good looking lines, including opcity when having the mouse on a line, inspiration has been retrieved from Zakaria Chowdhury (see: https://codepen.io/zakariachowdhury/pen/JEmjwq).
